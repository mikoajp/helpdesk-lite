# Design Tokens

Design tokens are the visual design atoms of the design system. They are named entities that store visual design attributes.

### Gray Scale

<div style={{display: 'flex', gap: '8px', marginBottom: '24px', flexWrap: 'wrap'}}>
  {[50, 100, 200, 300, 400, 500, 600, 700, 800, 900].map(shade => {
    const colors = {
      50: '#fafafa', 100: '#f5f5f5', 200: '#eeeeee', 300: '#e0e0e0',
      400: '#bdbdbd', 500: '#9e9e9e', 600: '#757575', 700: '#616161',
      800: '#424242', 900: '#212121'
    };
    const textColor = shade >= 500 ? 'white' : 'black';
    return (
      <div key={shade} style={{
        padding: '12px',
        backgroundColor: colors[shade],
        color: textColor,
        borderRadius: '4px',
        fontSize: '12px',
        minWidth: '80px',
        textAlign: 'center'
      }}>
        <strong>{shade}</strong><br/>
        {colors[shade]}
      </div>
    );
  })}
</div>

## üíæ Usage in Code

### TypeScript

\`\`\`typescript
import { ColorTokens, SpacingTokens, TypographyTokens } from '@shared/design-tokens';

// Access tokens
const primaryColor = ColorTokens.primary.main;
const spacing = SpacingTokens.md;
const fontSize = TypographyTokens.fontSize.base;
\`\`\`

### SCSS/CSS

\`\`\`scss
.my-component {
  // Using CSS custom properties
  color: var(--color-primary);
  padding: var(--spacing-md);
  font-size: var(--font-size-base);
  border-radius: var(--radius-sm);
}
\`\`\`
