## ðŸŽ¨ Design Tokens

Our design system is built on a foundation of **design tokens** - reusable values that define the visual language:

### Colors
- **Primary**: #3f51b5 (Indigo)
- **Status Colors**: Open (Blue), In Progress (Orange), Resolved (Green), Closed (Gray)
- **Priority Colors**: Low (Green), Medium (Orange), High (Red)
- **Semantic**: Success, Warning, Error, Info

### Spacing
- XS: 4px
- SM: 8px
- MD: 16px
- LG: 24px
- XL: 32px
- XXL: 48px

### Typography
- Font Family: Roboto, Helvetica Neue, sans-serif
- Font Sizes: 12px - 36px
- Font Weights: 300 (Light), 400 (Normal), 500 (Medium), 700 (Bold)

### Border Radius
- SM: 4px
- MD: 8px
- LG: 16px
- Full: 9999px (circles)
